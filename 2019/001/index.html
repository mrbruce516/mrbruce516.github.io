你好 😃 这里是爱摸鱼的坏船长，很开心你能找到这里～

  今天呢 算是也完成了自己的一个梦想，就是建立一个自己的网站，记录我的生活，虽然这个网站由于经费问题，并没有真正意义上的从 租借服务器 -> 建站 -> 购买域名、dns、cdn 等等，但是感谢科技的进步吧，GitHub 允许个人搭建博客网页在他们的服务器上，就这样误打误撞的也算是达成了自己的梦想 +1。

  后端用的是 hugo，这个工具真的还蛮好用的，官方说是世界上搭建静态网页最快的方式，的确非常快，可扩展性也挺强的，希望接下来的时间里慢慢美化自己的 blog。

  有地方记录自己的生活和成长正是我想要的，特别是以 markdown 作为创作工具，以后有精力的话想写写关于
生活，游戏，技术 等的文章啦～

  さあ～ 还请多多指教啦 (´･ω･`)
